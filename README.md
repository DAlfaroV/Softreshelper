# Softreshelper
WoW wotlk Addon used to display items soft reserved(via CSV from https://softres.it ) in game.

-------------------------
## Usage

- Use command /srs to display window.
- On main frame import CSV data from https://softres.it
- Click import to upload the data.
- Button View Res print reserved items and the player who reserved it in chat (Only for you)
- Clear Data clears the data(duh).
